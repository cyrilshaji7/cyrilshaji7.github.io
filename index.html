My Projects
Aqua Pinion - Autonomous Drowning Detection Band.
Using the power of machine learning and the portability of an arduino, me and my team has succeeded in creating a watch which can use various parameteres from your body to determine with a precision of 98% whether you are drowning in real time using a machine learning algorithm which runs on an arduino nano.

I recruited a support vector machine for drowning detection which relies on input data like body motion, pressure, and more sensors. By learning from a dataset containing drowning and non-drowning scenarios, the SVM model can accurately identify signs of distress in real-time, and could potentially save lives.

VisiSecure - Face Recognizing Door Security Mobile Application.
A smart camera equipped with advanced face recognition technology can instantly detect visitors at your door and send a notification. If the visitor is recognized as a familiar face, the camera will grant them access. If it's an unknown face, an alert will be sent to a mobile app developed using Flutter, ensuring a seamless and secure access control system.

Combined Python's OpenCV for image processing and Flutter's UI capabilities to create a powerful cross-platform solution for building intelligent, real-time camera applications, which will make homes safer for all of us.

GravityCanvas - Simulation Of Celestial Bodies In 2D.
GravityCanvas is a C++-powered N-Body 2D simulation web app that accurately models celestial body interactions and gravitational forces. Using efficient algorithms like Barnes-Hut, it tracks and predicts the positions and velocities of multiple objects, providing a robust simulation tool for astronomers and astrophysicists.

With a user-friendly interface, this web app lets users define celestial body parameters like mass and initial conditions, while the C++ backend handles complex calculations, ensuring real-time or near-real-time simulations. GravityCanvas empowers users to explore celestial system dynamics, making it a valuable resource for studying everything from planetary orbits to galactic interactions, accessible online.
